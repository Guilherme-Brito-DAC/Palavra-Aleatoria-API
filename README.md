# API de palavras aleatorias

API que fornece uma palavra aleatoria do dicionário pt-br.

## Métodos

`/Palavra`

```json
{
    "Idoso"
}
```

`/Palavra?inicial=q`

```json
{
    "Quebra"
}
```

`/Palavra?letras=4`

```json
{
    "Gato"
}
```

`/Palavra?letras=7&inicial=G`

```json
{
    "Goleada"
}
```
